# Six Seven File System

> [!CAUTION]
> **Disclaimer:** This is not a serious project. It is not secure in any way, it is extremely slow, it uses a lot of file space, and it uses a lot of RAM. It should not be used in production.

A file system using 6 7

Works with most file types!

## Tested filetypes:

_If a file format is not on this list, it does not mean it will not work. These are only the ones I have tested and confirmed._

#### Text:

    .txt
    .md

#### Code:

    .py
    .html
    .js

#### Image:

    .png
    .gif

Feel free to make a pull request adding to this list!
